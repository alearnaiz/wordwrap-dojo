# Coding Dojo: Word Wrap Kata

Implementación de la [Kata Word Wrap](http://codingdojo.org/kata/WordWrap/) por @danipise y @pere_jover en el Coding Dojo de [Mallorca Software Craftsmanship](https://www.meetup.com/Mallorca-Software-Craftsmanship/events/248657217/) 

El ejercicio consiste en implementar una función llamada wrap con dos parámetros: texto y tamaño de columna.
La función devuelve el texto, pero colocando saltos de línea de tal manera que no haya ninguna línea con un tamaño superior al de la columna.
